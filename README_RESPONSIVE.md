## ✨ Special Features

### 🎯 Responsive Breakpoints
```jsx
const responsive = useResponsive();

responsive.isMobile          // < 640px
responsive.isTabletSm        // 640-1024px
responsive.isTablet          // 1024-1280px
responsive.isDesktop         // >= 1280px
responsive.width             // Current width
responsive.height            // Current height
```

### 🎨 CSS Class System
```jsx
// Container
className="dashboard-container"

// Grids
className="summary-row"       // 1→2→3 columns
className="category-grid"     // Auto-fill grid

// Cards
className="summary-card"
className="category-card"

// Forms
className="filter-input"
className="filter-select"
className="download-btn"

// Tables
className="responsive-table"
className="table-card-list"   // Mobile view
className="table-wrapper"     // Desktop view
```

### 🎭 Conditional Rendering
```jsx
{responsive.isMobile ? <MobileView /> : <DesktopView />}
```
